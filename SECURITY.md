# Security Policy

## Supported Versions

The `Stablecoin Payment Plugin` project is currently in pre-release development.

At this stage, only the latest tagged release receives security updates. The `main` branch may
contain unstable or experimental changes and should not be used in production environments without
your own review and testing.

| Version                  | Supported          |
| ------------------------ | ------------------ |
| Latest 0.x release       | :white_check_mark: |
| Older 0.x releases       | :x:                |
| Unreleased / main branch | :x:                |

> Pre-1.0 versions are considered unstable and may introduce breaking changes.

## Reporting a Vulnerability

**Please do not report security vulnerabilities through public GitHub issues.**

Security is a top priority on this project. If you discover a vulnerability, please report it
privately via:

[okpainmoandrew@gmail.com](mailto:okpainmoandrew@gmail.com)

Please include:

- A clear description of the vulnerability
- Steps to reproduce, if applicable
- A proof of concept, if safe to share
- Potential impact assessment
- Affected module, chain, contract, backend service, or integration surface

We aim to acknowledge receipt within 48 hours and will keep you informed throughout the
investigation and remediation process.

We follow responsible disclosure practices and ask that you allow reasonable time to investigate and
release a fix before disclosing the issue publicly.
